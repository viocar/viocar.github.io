PERSISTENT CHARACTER STATS (PCS):

Length: 0x118

0x0 (b): This is a bitfield and I think it's dynamically modified by the game. Don't change it.
0x1 (b): Portrait. These IDs do not match the IDs in-game. Here's the table for how they line up with the standard portrait order:

02 00 03 01 (main portraits)
06 04 07 05 (alternates)

0x2 (b): Guild order.
0x3 (b): Current level.
0x4 (w): Main class. I have no idea why this is a word. Monsters will get 0xD.
0x8 (w): Sub class. Is 0xFFFFFFFF if no subclass.
0xC (h): Equip slot 1.
0xE (b): Forge slots. This includes ones that are filled by forges, but does not include forges that are innate on the gear.
0xF - 0x17 (b): Forges. List of forges:

01	ATK

An item can have nine forges.

0x18 - 0x23 (h/b): Equip slot 2 and forges.
0x24 - 0x2F (h/b): Equip slot 3 and forges.
0x30 - 0x3B (h/b): Equip slot 4 and forges.
0x3C - 0x4F (w/h): Appears to be the number of stats added by books. Please note that this value is added to your stats and is not the stat after modification. For instance, if your character has 0x32 base STR and one STR book, the value in the STR field here will be 0x1, not 0x33.

0x5E (h): LUC
0x60 (h): TEC
0x62 (h): "WIS" (always empty)

The second block begins at 0x64, and is a copy of block 1. Used for stat up passives in EO2 but is unused in EO3 (repurposed in EO4 for food buffs). However, it does apply HP Up and TP.

The third block begins at 0x78, and is a copy of block 2. It adds bonuses from equipment, and is automatically recalculated when you change equipment.

The fourth block begins at 0x8C. It is your current stats in battle. Outside of battle, I think this can show some weird values. Not sure why.

0xA0 (h): Current HP.
0xA2 (h): Current TP.
0xA4 (w): Current EXP (total gained since LV 1).
0xA8 - 0xBB: Name, in Shift-JIS. NOTE: Cap this to 9. The last halfword can be used for a character, but the game expects it to be a null byte. If you use the 10th character, and the next field is an ailment, the game can crash.
0xBC (h): Current ailment.
0xBE (b): Max level.
0xBF (b): Current limit.
0xC0 (b): Available stat points.
0xC1 (b): Internal ID.
0xC2 - 0xC9 (b): Common skill levels.
0xCA - 0xDC (b): Main class skill levels.
0xDD - 0xEF (b): Sub class skill levels.
0xF0 (h): Equipped limit.
0xF2 (b): Position (0-5).
0xF3 (b): Unknown - used in allocating skill points after a rest.
0xF4 (h): Unique sea quest ID. Should never show up in a save game.
0xF6 - 0xF9 (?) Unknown. Changing these values doesn't seem to do anything.
0xFA: Strength books read
0xFB: Vitality books read
0xFC: Agility books read
0xFD: Luck books read
0xFE: Technique books read
0xFF: Wis?? books read

Rest is unknown.
